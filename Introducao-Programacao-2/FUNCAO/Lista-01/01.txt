ser corrigido:
a) int quadrado(int x);
{
   return x * x;
}
b) int soma(int x, y) {
  int resultado;
     resultado = x + y;
     return resultado;
   }

     float a;

   }

a) O erro está no ';' que esta localizado na primeira linha, se remover o programa funcionara
